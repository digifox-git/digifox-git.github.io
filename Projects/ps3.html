 >>> My Projects >>> PlayStation®3 Game Guide

Below you will find a helpful and detailed-ish guide on installing more games onto your jailbrak'd PS3. This guide assumes you have Webman installed which can be verified by checking if the "webMAN Games" folder appears in the Game section of your XBM (PS3 Home Menu)

How do I install more games?


Step 1: Download the GAME from Vimm's Lair or any other fun file forest you choose. (Games should download in a FOLDER)


Step 2: Locate your PS3's IP through Settings >> Network Settings >> Settings and Connection Status List. Copy your IP address (You'll need this later!)


Step 3: Download FileZilla


Step 4: In FileZilla, navigate to File >> Site Manager, then press the "New Site" button


Step 5: After pressing the "New Site" button, copy these settings (Use the IP address you got in Step 2.) When you're done with that, just press "Connect"


Step 6: Drop your GAME (in FOLDER form) inside of your ps3 in the directory "/dev_hdd0/GAMES"

WARNING: DO NOT PUT THE GAME FOLDER ANYWHERE ELSE ON YOUR SYSTEM JUST TO BE SAFE (UNLESS YOU REALLY REALLY KNOW WHAT YOU'RE DOING!) 

Note: Having trouble seeing the image? Right click on it and select "Open Image In A New Tab" to enlarge... It's like magic!


Step 7: Hop over to your PS3 and go to Game >> webMAN Games >> webMAN Setup >> PS3 webMAN [Setup] and press "Refresh" at the top to refresh the games list


Step 8: Viola! Cello! Woohoo!!! You should have successfully downloaded the game onto your system. You can mount your lovely new game by navigating to Game >> webMAN Games >> PLAYSTATION®3 and selecting your game.


Step 9: THERE SHOULD NOT BE A STEP 9!!!
